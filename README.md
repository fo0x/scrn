# SmartShot — AI-асистент для скріншотів

Мобільний застосунок на Expo/React Native, який:
- читає скріншоти з телефону (папка `Screenshots`);
- розпізнає тип контенту;
- перетворює в корисну дію.

## Що вже реалізовано

1. **Доступ до медіатеки** та вибірка останніх скріншотів.
2. **Intent-класифікація** (демо-евристика + готова точка інтеграції для OCR/LLM).
3. **Побудова Smart Actions**:
   - пост із соцмереж → текстова чернетка поста;
   - нагадування/зустріч → подія календаря;
   - одяг/покупки → wishlist;
   - фільми/музика → media wishlist;
   - чеки → витрати;
   - квитки/подорожі → travel ideas.
4. **UI-панель** з групуванням автоматично створених дій.

## Архітектура

- `App.tsx` — головний екран та групування дій.
- `src/services/screenshotPipeline.ts` — пайплайн аналізу скріншотів.
- `src/types/domain.ts` — доменні моделі.
- `src/components/ActionCard.tsx` — картка дії.

## Ідеї для наступної версії

1. OCR + мультимодальна модель для реального вилучення тексту (OpenAI Vision).
2. Автозаповнення календарної події з датою/таймзоною/локацією.
3. Генерація кількох версій поста: короткий, експертний, дружній tone.
4. Детектор дублювання скріншотів (прибирати повтори).
5. Автоматичний канбан: "читати", "купити", "подивитись", "відповісти".
6. Пріоритизація дій за терміновістю (наприклад, рейс завтра).
7. Особистий граф знань: привʼязка людей, місць, брендів, інтересів.

## Запуск

```bash
npm install
npm run start
```

> Демо-версія використовує евристику за назвою/URI скріншота. Для production додайте OCR, embeddings та API-інтеграцію.
